◆ sht3x_read()

int16_t sht3x_read ( struct sht3x_sensor_data * sensor)

Reads out the results of a measurement.

Reads out results of a measurement that was previously started by sht3x_measure(). If the measurement is still in progress, this function returns an error. Temperature is returned in [degree Celsius], multiplied by 1000, and relative humidity in [percent relative humidity], multiplied by 1000.

Parâmetros
[in,out]sensor: Sensor struct
Retorna
0 if the command was successful, else an error code.

formulas for conversion of the sensor signals, optimized for fixed point algebra: Temperature = 175 * S_T / 2^16 - 45 Relative Humidity = * 100 * S_RH / 2^16
