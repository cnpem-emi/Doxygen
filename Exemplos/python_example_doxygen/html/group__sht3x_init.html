int16_t sht3x_probe ( struct sht3x_sensor_data * sensor)

Detects if a sensor is connected by reading out the ID register.

If the sensor does not answer or if the answer is not the expected value, the test fails.

Parâmetros
[in]sensor: Sensor Struct
Retorna
